EcoTide SwiftUI simulation

Motion-driven environmental simulation with reviewer-friendly telemetry.

EcoTide is a simulation you understand by watching the scene respond, not by reading a fake browser clone. This page gives reviewers the contract, the scenario entry points, and the shortest route into the native proof.

What it shows

  • Gravity interaction driven by motion input.
  • Live telemetry for ice integrity, habitat risk, and system state.
  • Fallback mode when motion sensors are unavailable.
  • A review-pack mindset even in a simulation app.

Why the proof stays in the app

  • The meaningful proof is the rendered simulation reacting to motion input or fallback mode.
  • The CLI exists to preserve reviewer posture, not to stand in for the visual simulation.
  • This page keeps the repo legible without flattening the project into a web-only story.

Review path

  • Open the repo README.
  • Run the CLI fallback for the `ecotide-review-pack-v1` contract.
  • Use Xcode/simulator for the real rendering and motion experience.
